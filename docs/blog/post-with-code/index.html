For this tutorial, I plan to revisit main concepts of the Analysis of Variance (ANOVA) test. ANOVA tests estimate how a quantitative dependent variable changes according to the levels of one or more categorical independent variables. In other words, using ANOVA, one can determine whether there is a statistically significant difference in the means of the groups with regards to the dependent variable in question.

Note

As a rule of thumb, ANOVA tests are suitable for comparing three or more groups, while using t-tests are better for for comparing two groups. In our case, because there are three groups (High/Medium/Low income inequalit), ANOVA is a better option to go with.

To conduct ANOVA test with a real-world example, a tailored dataset for this purpose, consisting of World Bank Indicators alongside the number of terror events recorded in each country from the Global Terrorism Database (GTD), will be used. In this analysis, the dependent variable will be the number of terror events occured across countries between 1990-2020 and the independent variable will be the economic inequality levels across countries. Therefore, what will be inquired is to see whether there is a statistically significant difference across countries having different income inequality levels (High/Medium/Low) in terms of the number of terror events they have experienced.

Independent variables are the ones we, as researchers, would like to test whether they have an effect on the outcome such as the impact of poverty on school success.

Dependents variables are also called outcome variables. Taking the poverty and school success example, a student’s academic performance is a dependent variable and this can be influenced by a variety of factors such as his/her parents’ poverty, education, and etc.

Null and alternative hypotheses for this analysis are;
  • Null Hypothesis (\(H_{0}\)): Countries, having high/medium/low economic inequality, are equal in terms of the number of terror events occurred.
    • \(\mu_{High Economic Ineqaulity}\) = \(\mu_{Medium Economic Ineqaulity}\) = \(\mu_{Low Economic Ineqaulity}\)
  • Alternative Hypothesis (\(H_{1}\)): At least one type of country is statistically different from others in terms of the number of terror events experienced.

Main Assumptions of ANOVA

ANOVA tests have 4 main assumptions to be inquired on. They are;

Main Assumptions for ANOVA
Assumption Explanation
Variable Type For ANOVA tests, it is necessary to have a single continuous quantitative dependent variable (DV) and a qualitative independent variable (IV) with a minimum of two levels.
Independence ANOVA tests require that the observations within each group are independent of each other, and this condition can be verified based on the study design.
Normality ANOVA tests require the distribution of the residuals (the differences between the observed values and the predicted values) to be normal. In other words, the residuals should be normally distributed around zero. As a rule of thumb, for large sample sizes, normality is assumed to be satisfied. Note that, if normality is not met, the non-parametric test, Kruskal Wallis test, should be used instead of ANOVA.
Equality of Variances
(Homogeneity)		 In ANOVA tests, the variances of the groups should be equal. If Equality of Variances is not met, Welch ANOVA test should be used
Two ways to control normality assumption
  1. Via visual depiction -> using Histogram and/or Quantile-Quantile Plot (QQ Plot)
  2. Via statistical testing -> using shapiro.test
    • the Null Hypothesis (\(H_{0}\)) is the data follows a normal distribution, while the alternative hypothesis (\(H_{1}\)) rejects that.
    • In this scenario, having \(p-value>0.05\) will demonstrate that there is not enough evidence to reject the Null Hypothesis (\(H_{0}\)) and therefore this criterion is met.
Two ways to control homogeneity assumption
  1. Via visual depiction -> using Histogram and/or Dot Plot and/or Scale-Location plot from plot() in R.
  2. Via statistical testing -> using Levene’s test and others such as Barlett’s test and etc.
    • the Null Hypothesis (\(H_{0}\)) is variances within groups are equal, while the alternative hypothesis (\(H_{1}\)) is variances within groups are not equal.
    • In this scenario, having \(p-value>0.05\) will demonstrate that there is not enough evidence to reject the Null Hypothesis (\(H_{0}\)) and therefore this criterion is met.

Checking assumptions with real-world data

After introducing the dataset and the main premises of ANOVA test, let’s go over whether the ANOVA test we will be conducing satisfies the main assumptions.

Beginning with Variable Type, we have a single continuous quantitative dependent variable (DV), number of terror events for each country annually, and a qualitative independent variable (IV) with a minimum of two levels, the level of economic inequality. So, Variable Type assumption is satisfied.

With regards to the Independence, the events in our dataset are independent from the each other as it is not high likely that number of terror events in one country affects another and etc.

To check Normality assumption, there are two ways, either a visual inspection or a statistical test, to go with. The below panel tab shows both.

  • Histogram
  • Shapiro Test
hist(dataset$count)

shapiro.test(dataset$count)

    Shapiro-Wilk normality test

data:  dataset$count
W = 0.31233, p-value < 0.00000000000000022

With a p-value below 0.05 as a result of the Shapiro test, we can reject the null hypothesis that the data follows a normal distribution, which is not right. However, considering the sample size in our case, we can assume that the normality assumption is met.

As a last assumption, to check Homogeneity, we again have two ways: either a visual inspection or a statistical test. The below panel tab demonstrates the result for both.

  • Histogram
  • Dot Plot
  • Levene’s Test
dataset$economic_inequality <- factor(dataset$economic_inequality, 
                                                  levels = c("High Inequality", "Medium Inequality", "Low Inequality"))

ggplot(dataset, aes(x = count, fill = economic_inequality)) +
  geom_histogram(position = "identity", alpha = 0.6) +
  labs(x = "Value", y = "Frequency") +
  ggtitle("Histograms of Values by Inequality Status") +
  facet_wrap(~ economic_inequality, nrow = 1) +
  theme(legend.position = "none")

ggplot(dataset, aes(x = economic_inequality, y = count)) +
  geom_point()

library(car)
